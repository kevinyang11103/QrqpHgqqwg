## 前言

欢迎来到新冠疫苗预约小程序+SSM项目。本项目致力于为广大用户提供便捷的新冠疫苗预约服务，结合当前热门的SSM框架和微信小程序技术，实现了一款功能完善、用户体验优良的新冠疫苗预约系统。

## 内容介绍

本项目主要包括以下功能模块：用户注册/登录、疫苗预约、预约查询、个人信息管理、公告通知等。通过这些模块，用户可以轻松完成疫苗预约全流程，实时了解疫苗预约情况，提高疫苗接种效率。此外，系统还提供了后台管理功能，方便管理员进行疫苗信息管理、预约管理、用户管理等操作。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Springmvc、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
